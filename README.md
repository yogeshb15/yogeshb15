# Yogesh Bhardwaj - Portfolio Website Source Code

This repository contains the source code for my personal portfolio website, hosted live at [https://yogeshb15.github.io](https://yogeshb15.github.io).

## About Me

I am Yogesh Bhardwaj, a Research Fellow at Delhi Technological University pursuing a Ph.D. in Applied Mathematics, specializing in cosmology and matter creation theories. I also work as an LLM Senior Engineer and AI-Trainer, developing advanced mathematical AI models.

## Website Features

*   **Space-Themed Design:** A visually appealing, dark theme with a starry background reflecting my interest in cosmology.
*   **Responsive Layout:** Optimized for viewing on desktops, tablets, and mobile devices.
*   **Comprehensive Sections:**
    *   About Me
    *   Research Focus
    *   Professional Experience (Contract Work)
    *   Publications
    *   Skills (including Python, ML, MATLAB libraries, LaTeX, Cosmological Datasets)
    *   Education
*   **Technologies Used:** Pure HTML, CSS, and JavaScript, hosted using GitHub Pages.

## Repository Contents

*   `index.html`: The main HTML file containing the structure and content of the website.
*   `<style>` block in `index.html`: Contains all the CSS styling for the space theme.
*   `<script>` block in `index.html`: Contains JavaScript for dynamic elements (e.g., twinkling stars).
*   `picture.jpg`: My profile picture used on the website.
*   `README.md`: This file.
